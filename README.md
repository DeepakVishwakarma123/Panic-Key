# Panic Key 🚨

**One shortcut. Total cover. Instant camouflage for your browser.**

---

## What it does

Panic Key is a browser extension that helps you instantly camouflage your browsing activity with a single keyboard shortcut — perfect for those *"someone's coming"* moments.

---

## Features

- **Instant Profile Switch** — Opens camouflage tabs automatically
- **Closes all tabs** across all windows — except Incognito
- **Clears last 1 hour** of browsing history
- **Clears last 1 year** of cache, localStorage, IndexedDB, form data, download history
- **Custom Profiles** — Create your own with up to 5 URLs per profile
- **Profile rules** — Minimum 3 URLs required, can delete up to 2 URLs per profile

---

## Default Shortcut

**`Ctrl + Shift + 1`**

> Can be changed via Chrome/Opera Extensions page → Keyboard Shortcuts

---

## How to Install Locally

**Chrome:**
1. Open Chrome → go to `chrome://extensions`
2. Enable **Developer Mode** (top right)
3. Click **Load Unpacked**
4. Select the root folder of the extension
5. Extension will appear — pin it and use

**Opera:**
1. Open Opera → go to `opera://extensions`
2. Enable **Developer Mode** (top right)
3. Click **Load Unpacked**
4. Select the root folder of the extension
5. Extension will appear — pin it and use

---

## How to Use

1. Install the extension
2. Create your camouflage profile with safe URLs
3. Press **`Ctrl + Shift + 1`**
4. Done. You're innocent. 😄

---

## Limitations (V1)

- Search bar suggestions cannot be cleared — browser limitation
- Profile deletion not available — coming in V1.2
- Incognito windows are not affected
